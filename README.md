# Image Segmentation Using Graph Cuts

## Running
- In the last cell, modify the image_path as the path of the image you want to segment and run the cells

## Interactive Segment Usage
- Pressing key ‘f’ allows the user to mark the foreground part of the image using a brush.
- Pressing key ‘b’ allows the user to mark the background part of the image using a brush.
- Pressing key ‘c’ starts the segmentation.
- Pressing key ‘a’ allows the user to add seeds.
